# Bandit6--7

## Goal
The password for the next level is stored somewhere on the server and has all of the following properties:

owned by user bandit7
owned by group bandit6
33 bytes in size

## Concepts learned
how to check file type and its attributes.
## Commands used
-cat
-ls
-file


## Explaination 
This level makes the user to find a specific file using meta data . Check the file type of all the files in the home directory using file . for 
current directory and * wild cardd for checking all the files at once. with attributes size , group , and owner. Once the file is found we can use CAT to read the file data
